# 名词节

import { Hb, Hr, Hg, Hp } from '@site/src/components/HighlightComponents';

## 名词节 (形式名词 の/こと - 名词从句)

###### 名词节：从属于使用了形式名词的主节的节。

- 私は <Hg>坂道を歩くの</Hg>が嫌いです。
- <Hg>一人でこの荷物を運ぶの</Hg>は無理です。
- <Hg>今年の夏がこんなに暑いの</Hg>は温暖化の影響です。
- 日本人は<Hg>ご飯を食べるの</Hg>が速いです。
- <Hg>私が言ったこと</Hg>をちゃんとメモしてください。
- 将来の夢は <Hg>大きな会社の社長になること</Hg>です。
- 趣味は <Hg>クラシック音楽を聴くこと</Hg>です。

## 引用节 (と) 

- 彼女は<Hg>寂しいと</Hg>つぶやいた。
- 母は<Hg>今晩は料理を作らないと</Hg>言いました。
- 私は<Hg>日本経済は2020年のオリンピックまでは好景気が続くと</Hg>思います。
- あそこに<Hg>駐車禁止と</Hg>書いてあります。
- あそこに<Hg>車を止めないでくださいと</Hg>書いてあります。
- 犯人は<Hg>もう二度としませんから、許してくださいと</Hg>泣いて謝った。
- 上司は<Hg>一体どうしてこんなことになったんだろうと</Hg>首を傾げだ。

## 疑问节(か)

- <Hg>病院の電話番号が何番か</Hg>(を)知っていますか。
