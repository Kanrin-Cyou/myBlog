# て助动词：ていく・てくる

import { Hb, Hr, Hg, Hp } from '@site/src/components/HighlightComponents';

##### ていく・ていきます　～ 动作远离（去）
<br/>
##### てくる・てきます  　～ 动作接近（来）
<br/>

<Hr>从教室外视角</Hr>
- 授業が始まって、学生たちが教室に<Hb>入っていきます</Hb>。 ～ 进去了
- 授業が終わって、学生たちが教室から<Hb>出てきます</Hb>。　～ 出来了

<Hr>从教室内视角</Hr>
- 授業が始まって、学生たちが教室に<Hb>入ってきます</Hb>。 ～ 进来了
- 授業が終わって、学生たちが教室から<Hb>出ていきます</Hb>。 ～ 出去了

#### ていく・ていきます
病院へ行く途中で、お見舞いの花を買っていきましょう。 ～ 买了花再去。

#### てくる・てきます
ここへ来る時、駅で地図おもらってきて。〜　车站拿了地图再来。
あそこの食堂で食べてきます。　去吃个饭（还回来）
ちょっと待っていてください。すぐ近くの店で買ってきます。去店买（还回来）
