## 附属语-助词

<Hg>格助词</Hg>&ensp;&ensp;:主要附在<Hb>体言</Hb>上，表达「文节」之间的关系。<br/>
<Hg>副助词</Hg>&ensp;&ensp;:附在<Hb>各种词</Hb>上，以增加不同的含义。<br/>
<Hg>终助词</Hg>&ensp;&ensp;:主要附在<Hb>句末</Hb>上，表达各种意思。<br/>
<Hg>接续助词</Hg>:主要附在<Hb>用言</Hb>上，连接前后「文节」。

| 助词种类 | 内容 |
| -------- | ---------|
| 格助词   | が　の　を　に　へ と　より　から　で　や   |
| 副助词   | は　も　こそ　さえ　でも だって　しか　ばかり　など　まで  だけ　ほど　きり(ぎり)　くらい(ぐらい)　なり やら　か　だの　なんて　ずつ　とか　すら |
| 终助词   | な(なあ)　や　よ　わ　こと な　ぞ　ぜ　とも　か の　ね(ねえ)　さ　かしら　もの　ものか  |
| 接续助词 | ば　と　ても(でも)　から　ので が　けれど(けれども)　のに　て(で)　し ながら　たり(だり)　なり　つつ　ものの　ところで   |

### 接续助词

主要附在<Hb>用言</Hb>上，连接前后「文节」，<Hb>做文节的助词</Hb>。

- (日本の寿司は美味しい)<Hr>けれど</Hr>、値段が高いです。
- (課税を引いた)<Hr>ので</Hr>、会社を休みます。

| 功能 | 助词| 語例 |
| -------- | -------- | ---------|
| 假定的顺接 | <Hb>ば</Hb> <Hg>と</Hg>  |雨が降れ<Hb>ば</Hb>、外に出ない。|
| 确定的顺接 | <Hb>ば</Hb> <Hg>と</Hg> <Hb>て(で)</Hb> <Hg>ので</Hg> <Hb>から</Hb>  |雨が降った<Hb>ので</Hb>、外に出なかった。|
| 假定的逆接 | <Hb>と</Hb> <Hg>ても(でも)</Hg> <Hb>ところで</Hb>  |たとえ雨が降っ<Hb>ても</Hb>、外に出よう。  |
| 确定的逆接 | <Hb>て(で)</Hb> <Hg>ても(でも)</Hg> <Hb>が</Hb> <Hg>けれど(けれども)</Hg> <br/><Hb>のに</Hb> <Hg>ながら</Hg> <Hb>つつ</Hb> <Hg>ものの</Hg> |雨が降った<Hb>けれど</Hb>、外に出た。 |
| 并立的关系 | <Hb>ば</Hb> <Hg>て(で)</Hg> <Hb>が</Hb> <Hg>けれど(けれども)</Hg> <Hb>し</Hb> <Hg>たり(だり)</Hg> |北海道にも行った<Hb>し</Hb>、青森にも行った。  |
| 补助的关系 | <Hb>て(で)</Hb> |もう一度やっ<Hb>て</Hb>みる。  |

## 自立语-接续词

##### 接续词用于句子句首和句子中间。

|  |  |
|---|---|
|顺接| <Hb>それで</Hb> <Hg>そこで</Hg> <Hb>だから</Hb> <Hg>すると</Hg> <Hb>したがって</Hb> <Hg>ゆえに</Hg> <Hb>よって</Hb>|
|逆接| <Hb>しかし</Hb> <Hg>だが</Hg> <Hb>ところが</Hb> <Hg>けれど・けれども</Hg> <Hb>が</Hb> <Hg>だけど</Hg> <Hb>でも</Hb> <Hg>なのに</Hg>|
|累加・并立	| <Hb>そして</Hb> <Hg>それから</Hg> <Hb>なお</Hb> <Hg>しかも</Hg> <Hb>それに</Hb> <Hg>そのうえ</Hg> <Hb>また</Hb> <Hg>および</Hg> <Hb>ならびに</Hb>|
|说明・补足| <Hb>つまり</Hb> <Hg>すなわち</Hg> <Hb>なぜなら・なぜならば</Hb> <Hg>ただし</Hg> <Hb>もっとも</Hb>|
|对比・选择| <Hb>それとも</Hb> <Hg>あるいは</Hg> <Hb>または</Hb> <Hg>もしくは</Hg>|
|转换| <Hb>さて</Hb> <Hg>ところで</Hg> <Hb>では</Hb> <Hg>ときに</Hg>|

###### 接续词
- 日本の寿司は美味しいです。<Hb>けれど</Hb>、値段が高いです。
- 試験お日は、鉛筆を用意してください。<Hb>または</Hb>、ボールペンでもいいです。

###### 接续助词/并立助词
- (日本の寿司は美味しい)<Hb>けれど</Hb>、値段が高いです。
- 試験お日は、鉛筆<Hb>または</Hb>ボールペンを用意してください。
- (課税を引いた)<Hb>ので</Hb>、会社を休みます。

| 接尾   | 转换                  | 例子                            |
| ------ | --------------------- | ------------------------------- |
| さ     | 形容词/形容动词->名词 | 大きい->大きさ 華やか->華やかさ |
| たて   | 动词->名词            | 焼きます->焼きたて              |
| らしい | 名词->形容词          | 男->男らしい                    |
| にくい | 动词->形容词          | 歩きます->歩きにくい            |
| がち   | 名词->形容动词        | 病気->病気がち                  |
| そう   | 形容词->形容动词      | 美味しい->美味しそう            |
| がる   | 形容词->动词          | 欲しい->欲しがる                |
| びる   | 名词->动词            | 大人->大人びる                  |
