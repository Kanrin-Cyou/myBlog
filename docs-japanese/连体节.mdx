# 连体节(定语从句/分词)

类似于 定语从句/分词 结构。

###### 单词修饰名词
- 动词：<Hg>使った</Hg>コップは洗っておいてください。
- 形容词：<Hg>美味しい</Hg>料理が食べたいです。
- 形容动词：私は<Hg>静かな</Hg>ところに住んでいます。
- 名词：それは<Hg>日本の</Hg>カメラです。
- 连体词：<Hg>小さな</Hg>犬を飼っています。

###### 连体节修饰名词
- <Hg>三年前、京都で撮った</Hg><Hr>写真</Hr>です。
- <Hg>いつも私を励ましてくれる</Hg><Hr>伊原先生</Hr>を尊敬しています。
- <Hg>魚を焼いている</Hg><Hr>匂い</Hr>がします。

###### 同位语从句
- この金属は、<Hg>熱によって形が元に戻る</Hg><Hp>という</Hp><Hr>性質</Hr>を持っている。
- <Hg>訪日外国人が去年二千万人を突破した</Hg><Hp>という</Hp><Hr>記事</Hr>を読んだ。
