# て助动词：授受关系

import { Hb, Hr, Hg, Hp } from '@site/src/components/HighlightComponents';

## 名詞

### I.我方给予
###### Nをあげます
- 私は母の日に<Hr>母に</Hr><Hb>赤い花を</Hb><Hg>あげました</Hg>。
- 部長：山本くんはお正月に<Hr>お子さんに</Hr><Hb>お年玉のお金を</Hb><Hg>あげますか</Hg>。
###### Nを差し上げます 
- <Hr>同僚に</Hr><Hg>あげる</Hg>。この大きいのは<Hr>課長に</Hr><Hg>差し上げよう</Hg>と思って。
###### Nをやります
- 山本：<Hr>遊びに来た子供達に</Hr>は<Hg>あげます</Hg>が、<Hr>うちの子供に</Hr>は<Hg>やりません</Hg>よ。
- 私の旅行中、<Hr>花に</Hr><Hb>水を</Hb><Hg>やる</Hg>のを忘れないでね。

### II.我方接受
###### Nをもらいます
- 私は<Hr>友達に</Hr><Hb>旅行のお土産を</Hb><Hg>もらいました</Hg>。
- 誰かにもらったの？　<Hr>兄に</Hr><Hg>もらったんだ</Hg>。

###### Nをいただきます
- 祖母は<Hr>市長から</Hr><Hb>88歳のお祝いの品を</Hb><Hg>いただきました</Hg>。


### III.给予我方

- 結婚のお祝いに、叔母は<Hr>私に</Hr><Hb>着物を</Hb><Hg>くれました</Hg>。
- このネックレスは、国を出る時、友達が<Hg>くれたもの</Hg>です。

## 動詞

### I.我方给予（为他人做好意的行为）
###### Vてあげます
- パーティーの後、中山さんは<Hr>ハルさんを</Hr><Hb>家まで送っ</Hb><Hg>てあげました</Hg>。
- 高橋さんは<Hr>菅さんに</Hr><Hb>いいアルバイトを紹介し</Hb><Hg>てあげました</Hg>。
###### Vて差し上げます
- A：荷物、重そうですね。持ちましょうか。
- B：私は大丈夫ですから、<Hb>こちらのおばあさまの荷物を持って</Hb><Hg>差し上げてください</Hg>。
###### Vてやります
- 私は毎晩、子供が寝る時、<Hb>歌を</Hb><Hg>歌ってやります</Hg>。

### II.我方接受（接受好意的行为，有请求对方做某事的感觉）
###### Vてもらいます
- 私は朝起きられないので、いつも<Hr>母に</Hr><Hb>起こし</Hb><Hg>てもらいます</Hg>。
- 野菜をたくさん買ってので、<Hr>店の人に</Hr><Hb>家まで届け</Hb><Hg>てもらいました</Hg>。
- 子供の頃<Hr>母に</Hr><Hb>作っ</Hb><Hg>てもらった</Hg>人形を、今も大切にしています。
###### Vていただきます
- <Hr>高橋先生に</Hr><Hb>スピーチの作文を直し</Hb><Hg>ていただきました</Hg>。


### III.给予我方（我或自己的家人对他人的行为感到感谢。表示受害时，用被动句表达）
###### Vてくれます
- ゆりさんはとても親切で、私が困っているといつも<Hb>助け</Hb><Hg>てくれます</Hg>。
- 町田さんは電車の事故があったことを、すぐにメールで<Hb>知らせ</Hb><Hg>てくれました</Hg>。
- 私は掃除が下手です。でも時々友達が来て、<Hb>私の部屋を綺麗にし</Hb><Hg>てくれます</Hg>。
- 夫が<Hb>作っ</Hb><Hg>てくれる</Hg>お弁当はいつも美味しい。
- エアコンが壊れったので、買った店に電話した。店の人はすぐに<Hb>直しに来</Hb><Hg>てくれた</Hg>。
###### Vてくださいます
- 山田先生はいつもとてもいい<Hb>お話をし</Hb><Hg>てくださいます</Hg>。

#### 解释分析

- 菅さんが窓を閉めました。=> 不包含感情，叙述事实 
- 菅さんが窓を閉めてくれました。=> 感激之情 
- 菅さんが窓を閉められました。 => 感到不悦（受害）
<br/>
- 菅さんが写真を見せました。=> 不知道是给谁看 
- 菅さんが写真を見せてくれました。 => 知道是给我看
