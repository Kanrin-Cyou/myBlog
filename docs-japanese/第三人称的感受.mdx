# 第三人称的感受

import { Hb, Hr, Hg, Hp } from '@site/src/components/HighlightComponents';

###### 在日语中，主语是第三人称的句子，句尾不能直接使用“表达心情，感觉，意志的说法”。

希望：<Hg>ほしい</Hg><Hg>Vたい</Hg><br/>
意志：<Hg>Vよう</Hg><Hg>〜つもりだ</Hg><br/>
意向：<Hg>思う</Hg><Hg>考える</Hg><br/>
感情感觉的形容词：<Hg>うれしい</Hg><Hg>いやだ</Hg><Hg>痛い</Hg><Hg>ねむい</Hg>

######  因为是他人的想法观点，不能直接用肯定句型。

- ❌ 林さんはこの車がほしいです。
- ❌ 林さんは頭が痛いです。

## 解决方法1 加入揣测转述含义

#### 揣测
<div class="flex-container">
###### 林さんは
##### 車がほしい<br/>車が買いたい<br/>寂しい<br/>頭がいたい 
###### のです。<br/>ようです。<br/>らしいです。<br/>そうです。（传闻）<br/>と言っていますです。
</div>

#### 转述
<div class="flex-container">
###### 林さんは
##### 車を買おう<br/>車が買うつもりだ<br/>この車は高いと思う<br/>仕事がいやだ 
###### と言っていますです。
</div>


## 解决方案2 〜がる

##### 作用 接尾词将形容词变为动词 
<br/>
###### 某人现在的感受感情 （がっています）
- <Hr>弟は</Hr>オーストラリアの大学に<Hg>行きたがっています</Hg>。
- <Hr>あちゃん</Hr>がミルクを<Hg>ほしがって</Hg>泣いています。
- <Hr>妹は</Hr><Hg>恥ずかしがって</Hg>、知らない人とほとんど話をしません。

###### 一般的倾向/某人的习惯（がります）
- この頃、タバコの煙を<Hg>いやがる</Hg>人が多くなりました。
- 子供は他の子供が持っているものを<Hg>欲しがります</Hg>。

私がNほしいです。
妹がNほしがっています。

- <Hr>タンさんは</Hr>日本人の友達が<Hg>ほしい</Hg><Hg>と言っています</Hg>。
- <Hr>タンさんは</Hr>日本人の友達が<Hg>ほしがっています</Hg>。
- <Hr>大川さんは</Hr>小川さんに<Hg>会いたくない</Hg>と言っていますが、どうしたんでしょう。
- <Hr>大川さんは</Hr>小川さんに<Hg>会いたがらない</Hg>が、どうしたんでしょう。

|             |    原形 ❌         |   がっています              |   そうです（様態）      |
|-------------|-----------------|-----------------------------|-------------------------|
|   林さんは  |   車がほしい    |   車が欲しがっています。    |   車が欲しそうです。    |
|             |   車が買いたい  |   車を買いたがっています。  |   車が買いたそうです。  |
|             |   寂しい        |   寂しがっています。        |   寂しそうです。        |
|             |   仕事がいやだ  |   仕事が嫌がっています。    |   仕事がいやそうです。  |
