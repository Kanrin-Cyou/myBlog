# 条件

## 使用场景综合比较
###### 假设某人明天有计划出行，但天气不确定时：
- <Hb>もし晴れ<Hg>たら</Hg></Hb>、海に行きます。（たら：如果晴天，我会去海边。）
- <Hb>もし晴れれ<Hg>ば</Hg></Hb>、海に行きます。（ば：如果晴天，我会去海边。）
- <Hb>晴れ<Hg>たら</Hg></Hb>、海に行きます。（たら：晴天了之后，我会去海边。）
- <Hb>晴れたとし<Hg>ても</Hg></Hb>、仕事に行かなければならない。（ても：即使晴天，我也得去工作。）

###### 提建议时：
- <Hb>日本に行く<Hg>なら</Hg></Hb>、京都も訪れてみてください。（なら：如果去日本，请也去京都。）
- <Hb>日本に行く<Hg>ならば</Hg></Hb>、京都も訪れてください。（ならば：如果去日本，请也去京都。更正式）

###### 自然规律或习惯：
- <Hb>春になる<Hg>と</Hg></Hb>、桜が咲きます。（と：一到春天，樱花就会开。）


## 未来要发生

### 〜たら
###### 用法 
动词た形 + ら
###### 解释 
表示假设条件，常用于不确定的假设或未来可能发生的情况。
- <Hb>雨が降っ<Hg>たら</Hg></Hb>、<Hr>家にいます</Hr>。
如果要是下雨的话，我会呆在家。
###### 例子 
- <Hb>もし、来年の良いカレンダーがあっ<Hg>たら</Hg></Hb>、<Hr>買ってきてください</Hr>。
- <Hb>気分が悪かっ<Hg>たら</Hg></Hb>、<Hr>帰っても良いんですよ</Hr>。
- <Hb>そのカバンが高くなかっ<Hg>たら</Hg></Hb>、<Hr>私も一つ欲しいです</Hr>。
- <Hb>もし、体のことが心配だっ<Hg>たら</Hg></Hb>、<Hr>来ちゃんと健康診断を受けt方がいいですよ</Hr>。
- <Hb>もし、男の子だっ<Hg>たら</Hg></Hb>、<Hr>「あきら」という名前をつけましょう</Hr>。
- <Hb>明日いい天気でなかっ<Hg>たら</Hg></Hb>、<Hr>うちにいたいです</Hr>。

### 〜Vたら
###### 用法 
动词た形 + ら
###### 解释 
与基本「〜たら」使用相同，但特别强调动词处于完成状态的假设条件。
- <Hb>宿題が終わっ<Hg>たら</Hg></Hb>、<Hr>遊びに行きます</Hr>。 做完作业后，我会出去玩。
###### 例子 
- <Hb>六月になっ<Hg>たら</Hg></Hb>、<Hr>山登りに行きませんか</Hr>。
- <Hb>駅に着い<Hg>たら</Hg></Hb><Hr>連絡してください</Hr>。5分で改札口に迎えに行きます。
- <Hb>行くか行かないか、決まっ<Hg>たら</Hg></Hb>、<Hr>私にも知らせてください</Hr>。
- <Hb>卒業し<Hg>たら</Hg></Hb>、<Hr>国へ帰るつもりです</Hr>。
- <Hb>もうすぐ田中さんがきますよ。田中さんが来<Hg>たら</Hg></Hb>、<Hr>会議を始めましょう</Hr>。

### 〜と
###### 用法 
普通型的现在型 + と
###### 解释 
表示恒常条件，一件事必然导致另一件事, 常适用于自然规律或习惯。
- <Hb>春になる<Hg>と</Hg></Hb>、<Hr>桜が咲きます</Hr>。 一到春天，樱花就开了。
###### 例子 
- <Hb>山では１１月になる<Hg>と</Hg></Hb>、<Hr>雪が降ります</Hr>。
- <Hb>このボタンを押さない<Hg>と</Hg></Hb>、<Hr>お釣りが出ません</Hr>。
- <Hb>部屋の電気が暗い<Hg>と</Hg></Hb>、<Hr>本が読めません</Hr>。
- <Hb>周りが静かだ<Hg>と</Hg></Hb>、<Hr>よく寝られます</Hr>。
- <Hb>十八歳以上でない<Hg>と</Hg></Hb>、<Hr>入れません</Hr>。
- <Hb>この道を右に曲がる<Hg>と</Hg></Hb>、<Hr>郵便局があります</Hr>。
- <Hb>気をつけない<Hg>と</Hg></Hb>、<Hr>怪我をするよ</Hr>。

## 如果这样，就

### 〜ば（假设条件）
###### 用法 
动词的假定形（〜ば）、形容词去い加ければ
###### 解释 
表示假设条件，强调一种【客观的或自然的因果关系】，比〜たら更为正式。
- 早く寝れば、健康にいいです。 如果早点睡觉，对健康有好处。
- ❌ もし熱が出れば、この薬を飲んでください。
- ✅ もし熱が出たら、この薬を飲んでください。

###### 例子 
- <Hb>軽い風邪でしょう。ゆっくり休め<Hg>ば</Hg></Hb>、<Hr>治りますよ</Hr>。
- <Hb>市役所でもらったお知らせの文は難しいです。よく読まなけれ<Hg>ば</Hg></Hb>分<Hr>かりません</Hr>。
- <Hb>料理が美味しけれ<Hg>ば</Hg></Hb>、<Hr>そのレストランに食べに行きたい</Hr>。
- <Hb>このDVD、とてもいいよ。見たけれ<Hg>ば</Hg></Hb>、<Hr>貸してあげるよ</Hr>。
- <Hb>明日暇<Hg>なら</Hg></Hb>、山の写真の展覧会に行きませんか。
- <Hb>部屋が静かでなけれ<Hg>ば</Hg></Hb>、<Hr>私は勉強できません</Hr>。
- <Hb>もし、その人がいい人なら<Hg>ば</Hg></Hb>、<Hr>一緒に仕事をしたい</Hr>。

### 〜なら（假设 / 建议）
###### 用法 
名词 / 形容动词 / 动词普通形 + なら
###### 解释 
表示假设条件或提出建议，通常用于对前述句子的回应。
###### 例子 
- 日本に行くなら、新幹線に乗ってみてください。
- 日本に来たら、是非鹿児島にお立ち寄りください。

### 〜ならば（正式假设 / 建议）
###### 用法 
名词 / 形容动词 / 动词普通形 + ならば
###### 解释 
与「〜なら」类似，但更正式。
###### 例子 
日本へ行くならば、京都も訪れてください。

### 〜ても（让步条件）
###### 用法 
动词て形 + も、形容词て形 + も
###### 解释 
表示让步条件，即使前项成立，后项的结果也不会改变。
###### 例子 
雨が降っても、試合はあります。
即使下雨，比赛也会进行。
