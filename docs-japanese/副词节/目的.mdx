# 目的

### ~に

表示移动到某地是为了做某事

- 明日、空港へ母を迎え<Hb>に</Hb>行きます。
- 昼休みには、寮へ食事<Hb>に</Hb>戻ります。
- 連休には京都へお寺の見物<Hb>に</Hb>行きます。 

### Vる＋ため(に)　 in order to...

表示行为的目的，强烈的意志，前后主语必须一致。

- デザインを勉強する<Hb>ために</Hb>、専門学校に行っています。
- 田中さんは走る力をつける<Hb>ため</Hb>、毎日１０キロ走っています。
- 留学の<Hb>ために</Hb>準備しなければならないことがたくさんあります。
- この会を何の<Hb>ために</Hb>作ったのか、目的をもう一度考えてみよう。

- 夕日を見<Hb>に</Hb>海へ行った。
- 事故の原因を調べる<Hb>ために</Hb>海へ行った。

### ように (期待)
