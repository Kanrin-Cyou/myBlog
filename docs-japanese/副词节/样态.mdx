# 样态/方式

### Vている

表示状态，正在进行中。一般情况下使用自动词，

- 事故がありました。地下鉄が止まっ<Hb>ています</Hb>。
- カバンが開い<Hb>ています</Hb>。


- 赤ちゃんたちはもう目を開け<Hb>ています</Hb>。(他动词)


た 动作结果的状态

- メガネをかけている人＝メガネをかけた人

### Vてあります

人有目的地进行某种行为的结果存续状态。使用他动词。

-　母の部屋には家族の昔の写真が飾ってあります。
-　このノート、誰のでしょうか。　りんさんのですよ。裏にりんさんの名前が書いてあります。
-　お迎えに来ました。（私が）門の前に私の車を止めてありますから、すぐに出発しましょう。撮影現場には着く時間を知らせてあります。
-  旅行は来週ですね。もう準備はしてありますか。　ええ、見物するコースを決めてあります。レストランも予約してあります。
-　私はもう夏休みの計画表を作ってあります。
