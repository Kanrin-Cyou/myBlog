# 原因

<Hg>日本のドラマを見たいから</Hg>、日本語を<Hr>勉強しています</Hr>。

## 〜から (主观要求)
###### 解释
<Hp>【主观要求】</Hp>表示说话人的希望，后多接意志（〜たいです・つもりです）或者指示（〜なさい・〜てください）。
###### 用法
丁寧形・普通形+から
###### 例子
<br/>
##### 副词节
- <Hb>暑い<Hg>から</Hg></Hb>、<Hr>気をつけてね</Hr>。
- <Hb>お腹がいっぱいだ<Hg>から</Hg></Hb>、<Hr>食べたくないの</Hr>。
- <Hb>電車が来ます<Hg>から</Hg></Hb>、<Hr>ご注意ください</Hr>。
- <Hr>この箱、捨てないてね</Hr>。<Hb>後で使う<Hg>から</Hg></Hb>。

##### 非副词节
- <Hb>ちょっとでいい<Hg>から</Hg></Hb>ヒントを教えてくださいよ。
- A: どうして毎年富士山に登るんですか。 B:<Hb>富士山が好きだ<Hg>から</Hg></Hb><Hr>です</Hr>。


## 〜ので (客观解释)
###### 解释
<Hp>【客观解释】</Hp>后接表示结果趋势的原因，柔软请求辩解。
###### 用法
普通形＋ので｜名词化の+で
###### 例子
- <Hb>ネットにレストランの地図があった<Hg>ので</Hg></Hb>、<Hr>印刷しました</Hr>。
- すみません。<Hb>ちょっと寒い<Hg>ので</Hg></Hb>、<Hr>窓を閉めてくださいませんか</Hr>。
- <Hb>音楽が好きな<Hg>ので</Hg></Hb>、<Hr>よくコンサートに行きます</Hr>。
- 冬休みに家族に会いました。<Hb>みんな、元気だった<Hg>ので</Hg></Hb>、<Hr>安心しました</Hr>。
- <Hb>明日は休みな<Hg>ので</Hg></Hb>、<Hr>友達とサッカーをします</Hr>。
- <Hb>調べています<Hg>ので</Hg></Hb>、<Hr>少しお待ちください</Hr>。

## Vて/ Vなくて (个人事实观点意见)
###### 解释
<Hp>【个人事实观点意见】</Hp>表示直接原因或理由，用于表示一个动作或状态引起了某种结果。常用于描述情绪或情感的动词后。
###### 用法
て型
###### 例子
- <Hb>疲れ<Hg>て</Hg></Hb>、<Hr>もう動けない</Hr>。（因为累了，所以已经无法动了）
- <Hb>お金がなく<Hg>て</Hg></Hb>、<Hr>新しい服が買えない</Hr>。（因为没钱，所以买不起新衣服）
- <Hb>前に大きい人がい<Hg>て</Hg></Hb>、<Hr>よく見えません</Hr>。
- <Hb>お話でき<Hg>て</Hg></Hb>、<Hr>嬉しかったです</Hr>。
- <Hb>声は小さく<Hg>て</Hg></Hb>、<Hr>よく聞こえません</Hr>。
- <Hb>母の病気を心配<Hg>で</Hg></Hb>、<Hr>眠れませんでした</Hr>。
- <Hb>夜遅くまで仕事が終わらなく<Hg>て</Hg></Hb>、<Hr>大変でした</Hr>。
- 歯が痛かったので、<Hb>ご飯が食べられなく<Hg>て</Hg></Hb><Hr>困ります</Hr>。

## 〜し (种种理由原因)
###### 解释
<Hp>【种种理由原因】</Hp>即便只列举一种，隐含也还是有其他原因。用于强调“既…又…”的情况。
###### 用法
普通形+し
###### 例子
- 今日は <Hb>雨も降っている<Hg>し</Hg></Hb>、<Hr>出かけないで家でビデオを見ましょう</Hr>。
-  <Hb>今のアパートは狭い<Hg>し</Hg></Hb>、 <Hb>不便だ<Hg>し</Hg></Hb>、<Hr>もっといいところに住みたいです</Hr>。
-  <Hb>掃除もした<Hg>し</Hg></Hb>、<Hb>料理も作った<Hg>から</Hg></Hb>、<Hr>駅まで友達を迎えに行こう</Hr>。

## 〜ため(に) (书面)
###### 解释
<Hp>【书面】</Hp>表示原因或目的。用于正式场合或书面语。可以表示既可以是正面的目标，也可以是负面的原因。
###### 用法
普通形＋名词
###### 例子
- <Hb>事故の<Hg>ため</Hg></Hb>、<Hr>電車が遅れています</Hr>。
- <Hb>出席者が少なかった<Hg>ため</Hg></Hb>、<Hr>少し早く終わった</Hr>。
- <Hb>台風が近づいている<Hg>ため</Hg></Hb>、<Hr>中止になった</Hr>。
