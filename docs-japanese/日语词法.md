## 附属语-助词

### 助词分类

1. 格助词:主要附着在<Hb>体言</Hb>上，表达「文节」之间的关系。
2. 接续助词:主要附在<Hb>用言</Hb>上，连接前后「文节」。
3. 副助词:附在<Hb>各种词</Hb>上，以增加不同的含义。
4. 终助词:主要附在<Hb>句末</Hb>，表达各种意思。

| 助词种类 | 内容 |
| -------- | ---------|
| 格助词   | が　の　を　に　へ と　より　から　で　や   |
| 接续助词 | ば　と　ても(でも)　から　ので が　けれど(けれども)　のに　て(で)　し ながら　たり(だり)　なり　つつ　ものの　ところで   |
| 副助词   | は　も　こそ　さえ　でも だって　しか　ばかり　など　まで  だけ　ほど　きり(ぎり)　くらい(ぐらい)　なり やら　か　だの　なんて　ずつ　とか　すら |
| 终助词   | な(なあ)　や　よ　わ　こと な　ぞ　ぜ　とも　か の　ね(ねえ)　さ　かしら　もの　ものか  |

### 格助词

主要附着在<Hb>体言</Hb>上，表达「文节」之间的关系。

### 接续助词

主要附在<Hb>用言</Hb>上，连接前后「文节」，<Hb>做文节的助词</Hb>。

- (日本の寿司は美味しい)<Hr>けれど</Hr>、値段が高いです。
- (課税を引いた)<Hr>ので</Hr>、会社を休みます。

| 功能 | 助词| 語例 |
| -------- | -------- | ---------|
| 假定的顺接 | <Hb>ば</Hb> <Hc>と</Hc>  |雨が降れ<Hb>ば</Hb>、外に出ない。|
| 确定的顺接 | <Hb>ば</Hb> <Hc>と</Hc> <Hb>て(で)</Hb> <Hc>ので</Hc> <Hb>から</Hb>  |雨が降った<Hb>ので</Hb>、外に出なかった。|
| 假定的逆接 | <Hb>と</Hb> <Hc>ても(でも)</Hc> <Hb>ところで</Hb>  |たとえ雨が降っ<Hb>ても</Hb>、外に出よう。  |
| 确定的逆接 | <Hb>て(で)</Hb> <Hc>ても(でも)</Hc> <Hb>が</Hb> <Hc>けれど(けれども)</Hc> <br/><Hb>のに</Hb> <Hc>ながら</Hc> <Hb>つつ</Hb> <Hc>ものの</Hc> |雨が降った<Hb>けれど</Hb>、外に出た。 |
| 并立的关系 | <Hb>ば</Hb> <Hc>て(で)</Hc> <Hb>が</Hb> <Hc>けれど(けれども)</Hc> <Hb>し</Hb> <Hc>たり(だり)</Hc> |北海道にも行った<Hb>し</Hb>、青森にも行った。  |
| 补助的关系 | <Hb>て(で)</Hb> |もう一度やっ<Hb>て</Hb>みる。  |

## 自立语-接续词

接续词用于句子句首和句子中间。

|  |  |
|---|---|
|顺接| <Hb>それで</Hb> <Hc>そこで</Hc> <Hb>だから</Hb> <Hc>すると</Hc> <Hb>したがって</Hb> <Hc>ゆえに</Hc> <Hb>よって</Hb>|
|逆接| <Hb>しかし</Hb> <Hc>だが</Hc> <Hb>ところが</Hb> <Hc>けれど・けれども</Hc> <Hb>が</Hb> <Hc>だけど</Hc> <Hb>でも</Hb> <Hc>なのに</Hc>|
|累加・并立	| <Hb>そして</Hb> <Hc>それから</Hc> <Hb>なお</Hb> <Hc>しかも</Hc> <Hb>それに</Hb> <Hc>そのうえ</Hc> <Hb>また</Hb> <Hc>および</Hc> <Hb>ならびに</Hb>|
|说明・补足| <Hb>つまり</Hb> <Hc>すなわち</Hc> <Hb>なぜなら・なぜならば</Hb> <Hc>ただし</Hc> <Hb>もっとも</Hb>|
|对比・选择| <Hb>それとも</Hb> <Hc>あるいは</Hc> <Hb>または</Hb> <Hc>もしくは</Hc>|
|转换| <Hb>さて</Hb> <Hc>ところで</Hc> <Hb>では</Hb> <Hc>ときに</Hc>|

<Hb>接续助词</Hb>和<Hb>并立助词</Hb>同时也可以作为<Hr>接续语</Hr>, 但是这两种助词，不可以用于句首。
<br/>
<br/>

接续词
- (日本の寿司は美味しい)<Hr>けれど</Hr>、値段が高いです。
- (課税を引いた)<Hr>ので</Hr>、会社を休みます。

接续助词/并立助词
- 日本の寿司は美味しいです。<Hb>けれど</Hb>、値段が高いです。
- 試験お日は、鉛筆<Hb>または</Hb>ボールペンを用意してください。
- 試験お日は、鉛筆を用意してください。<Hb>または</Hb>、ボールペンでもいいです。

## 接辞-接尾词
`山田さん`
`靴屋`
`三人`
`私だち`
`風邪っぽい`

| 接尾   | 转换                  | 例子                            |
| ------ | --------------------- | ------------------------------- |
| さ     | 形容词/形容动词->名词 | 大きい->大きさ 華やか->華やかさ |
| たて   | 动词->名词            | 焼きます->焼きたて              |
| らしい | 名词->形容词          | 男->男らしい                    |
| にくい | 动词->形容词          | 歩きます->歩きにくい            |
| がち   | 名词->形容动词        | 病気->病気がち                  |
| そう   | 形容词->形容动词      | 美味しい->美味しそう            |
| がる   | 形容词->动词          | 欲しい->欲しがる                |
| びる   | 名词->动词            | 大人->大人びる                  |
