# 并立节

and/or/but

###### て
- （日記）九月六日ー青木さんにメールを<Hg>送り</Hg>、12時ごろねた。（书面语）
- 太郎はケーキを買いに<Hg>行って</Hg>、花子はテーブルの上を片付けました。
- 花子はテーブルの上を<Hg>片付けて</Hg>、太郎はケーキを買いに行きました。

###### たり 动作状态的并列
- 休みの日は家でインターネット<Hg>したり</Hg>、近くの公園を散歩したりします。
- 休みの日は近くの公園散歩<Hg>したり</Hg>、家でインターネットしたりします。

###### し 理由或性质的并列
- 彼は頭もいい<Hg>し</Hg>、運動もできる。

###### ながら
- 歌を歌い<Hg>ながら</Hg>ピアノを弾くことができます。
- ピアノを弾き<Hg>ながら</Hg>歌を歌うことがでsきます。

###### ～しつつ 正式
- コーヒーを飲み<Hg>つつ</Hg>、仕事を進める。 （边喝咖啡边推进工作。）
- 歩き<Hg>つつ</Hg>考える。 （边走边想。）
- 反省し<Hg>つつ</Hg>も、同じ失敗を繰り返してしまう。（虽然反省着，但仍不断重复同样的错误。）
- 健康のことを気にし<Hg>つつ</Hg>も、つい夜更かししてしまう。（虽然注意健康，但总是不小心熬夜。）

常用固定搭配：
•	知りつつ（明知道，却…）
•	思いつつ（虽然这样想，但…）
•	感じつつ（虽然感觉到，但…）
•	気にしつつ（虽然在意，却…）
